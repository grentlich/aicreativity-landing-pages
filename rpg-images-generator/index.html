Comment ça marche

Une expérience en trois phases pour une immersion totale

1

Préparation de l'aventure

Avant la partie, le MJ utilise notre application web pour générer tous les éléments visuels nécessaires à son scénario.

  • Lieux, créatures, objets, cartes, PNJ
  • Style graphique cohérent avec l'univers du jeu
  • Stockage et organisation des images pour un accès facile
2

Création des personnages

Les joueurs accèdent à l'application pour générer le visuel de leur personnage.

  • Portraits détaillés selon les caractéristiques du personnage
  • Styles variés (fantasy, cyberpunk, steampunk...)
  • Personnalisation selon les caractéristiques du jeu
3

Interaction en temps réel

Une fois la partie lancée, le bot Discord permet de :

  • Diffuser facilement les images générées dans les channels dédiés
  • Générer à la volée de nouvelles images via des commandes Discord
  • Fluidifier la narration avec des visuels créés instantanément
